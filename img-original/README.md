# img-original — 원본 마스터 이미지

여기에는 **압축 전 원본 풀해상도 사진**(인쇄급, 예: 5400×7200)이 들어 있습니다.

- 용도: 재인쇄, 액자, 고해상도가 필요한 작업, 향후 재편집/재최적화 소스.
- **사이트(청첩장)는 이 폴더를 직접 쓰지 않습니다.** 실제로 로딩되는 것은 한 단계 위 `../img/` 의 최적화본입니다.
- 파일명·폴더 구조는 `../img/` 와 1:1로 동일합니다.

## 다시 최적화하려면
저장소 루트에서:

```bash
python3 optimize_images.py
```

이 스크립트는 항상 `img-original/` 을 소스로 읽어 `img/` 에 최적화본을 다시 생성합니다.
(긴 변 최대 1600px, 품질 86, EXIF 회전 보정)
